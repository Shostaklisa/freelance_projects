# Предсказание успешности стартапов и анализ факторов, влияющих на их развитие

**Цель проекта** — разработать модель машинного обучения для предсказания успешности стартапа (останется операционным или закроется), а также провести разведочный анализ данных и сформировать рекомендации для будущих основателей стартапов.

В рамках проекта были поставлены **задачи**:
1) Построить модель, определяющую статус стартапа: operating или closed.
2) Провести анализ данных и определить, какие факторы наиболее сильно влияют на успешность стартапа.

Этапы работы:
1. Загрузка данных и библиотек. Подготовка окружения и данных для анализа.

2. Предобработка данных:
- приведение типов, переименование колонок;
- обработку и заполнение пропусков;
- выявление и устранение явных и неявных дубликатов;
- исправление опечаток.

3. Исследовательский анализ данных (EDA)
Были изучены числовые и категориальные признаки, их распределения и взаимосвязи. 

4. Корреляционный анализ
Использовалась phik-корреляция — так как зависимости между признаками были в основном нелинейными.
Выявлены признаки с мультиколлинеарностью, удалены следующие столбцы:
city, state_code, year_first_funding, region, name, first_funding_at, last_funding_at.

5. Моделирование:
- Baseline
- Перебор моделей с гиперпараметрами с помощью RandomizedSearchCV:

DecisionTreeClassifier
KNeighborsClassifier
LogisticRegression
XGBClassifier
CatBoostClassifier

Лучшей оказалась CatBoostClassifier.
- Улучшение модели. Учли сильный дисбаланс классов, подбирая веса на минорный класс; переработали редкие категориальные значения; оценивали качество по F1, также анализировали precision и recall.

6. Оценка важности признаков

Использовались техники:
- Permutation Importance
- Feature importance
- shap

**Финальный результат**

Построена модель для предсказания статуса стартапов и сформированы рекомендации для основателей:

1) Факторы успеха стартапа:

- длительное время жизни проекта;

- поздние раунды финансирования (новые раунды → живой интерес инвесторов);

- расположение в крупных инновационных центрах (SF, NY, London и др.);

- наличие успешной экосистемы, крупных IT-компаний и венчурных фондов поблизости.

2) Факторы риска:

- малый lifetime;

- малое количество раундов финансирования;

- низкий общий объём привлечённых средств;

- расположение в регионах с низкой стартап-активностью.

*Предсказание на тестовой выборке*

На финальном этапе была использована лучшая модель CatBoostClassifier: выполнен прогноз статусов стартапов для тестового набора данных.
